# E‑Commerce Sales & Customer Feedback Analytics (SQL)

This project analyzes e‑commerce sales and customer feedback data using SQL.

## What this project shows
- Database design (normalized tables + ER diagram)
- Sales trend analysis and top‑selling products
- Customer segmentation by value and engagement
- Stored procedures and automated KPI reporting

## Files
- `E-commerce-sales-and-feedback-database.pdf`: Full project slides with SQL queries, ERD, and business insights.

## Tools used
- SQL (joins, aggregations, CASE)
- Relational database design (ER modeling)
- Sales and feedback analytics
